FR_Dashboard - je interaktivní dashboard s načtenými daty, který lze po stažení Power BI použít pro okamžitou práci s daty.

thesis - v této složce je obsažen zdrojový kód pro Latex, ve kterém byla práce vytvořena, a zněhož je vytvořeno i samotné PDF, jež je také součástí složky

DP-data - v této složnce jsou obsaženy veškeré soubory, které byly vytvořeny pro potřeby této diplomové práce.

	Original_Data_Sources - obsahuje originální datové sady, z nichž jsou brány potřebné hodnoty pro celkovou analýzu. Tata data jsou nahrána do Stage databáze.
	Google_Trans - obsahuje dílčí části datových sad stažených z Google trends za využití python skriptu.
	transform - obsahuje pomocné soubory sloužící pro následnou transformaci pomocí python skriptu.
	Converted_Data_Sources - obsahuje transformovaná data, kterou jsou následně náhrána do Target databáze.
	Pictures - obsahuje obrázky modelů, grafů a teoretických ukázek, které jsou použity v Diplomové práci.
	Python_scripts - obsahuje skripty, které byly vytvořeny pro potřeby diplomové práce (stahování dat, transformace dat, statistické vyhodnocení, ...)
	SQL - obsahuje sql soubory pro vytvoření a nastavení databází datového skladu.
	Pentaho_Transactions - obsahuje transakce pro jednotlivé části tvorby datového skladu (stage, transform, target).
	Pentaho_Jobs - obsahuje joby pro kopletní vytvoření a naplnění datového skladu a také částečné joby pro jednotlivé fáze.
	DP-TFR - zdrojový kód pro Enterprase Architect obsahující veškeré vytvořené modely.
